Proyecto Final de Angular

Información

En este proyecto se presentan dos funcionalidades principales.

1) Ver los Gifs del momento
2) Ver los Stickers del momento
3) Poder buscar gifs
4) Adicional una página 404 cuando se navega a una ruta inexistente.

Para el proyecto se utilizó el API de Giphy

Gifs del momento

Utilizando el menú podrá ir a la sección de Gifs y así ver los 50 gifs del momento.

Stickers del momento

Utilizando el menú podrá ir a la sección de Stickers y así ver los 50 stickers del momento.
